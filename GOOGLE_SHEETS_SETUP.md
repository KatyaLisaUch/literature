# Подключение Google Sheets

## 1. Создайте Google Таблицу

Вариант А:

1. Откройте Google Drive.
2. Создайте новую Google Таблицу.
3. Назовите ее `Учет сдачи стихов - база`.

Вариант Б:

1. Загрузите файл `google-sheets-base-template.xlsx` в Google Drive.
2. Откройте его как Google Таблицу.

## 2. Добавьте Apps Script

1. Откройте Google Таблицу.
2. Выберите `Extensions` -> `Apps Script`.
3. Удалите весь код из `Code.gs`.
4. Вставьте код из файла `google-apps-script/Code.gs`.
5. Нажмите `Save`.

## 3. Создайте листы и демо-учителя

1. В Apps Script выберите функцию `setupInitialData`.
2. Нажмите `Run`.
3. Разрешите доступы Google.

После этого будет создан учитель:

- логин: `teacher1`
- пароль: `123456`

Также будет создан админ-пароль для регистрации новых учителей:

- админ-пароль: `admin123`

## 4. Опубликуйте Web App

1. Нажмите `Deploy` -> `New deployment`.
2. Тип: `Web app`.
3. `Execute as`: `Me`.
4. `Who has access`: `Anyone`.
5. Нажмите `Deploy`.
6. Скопируйте URL, который заканчивается на `/exec`.

## 5. Подключите сайт

1. Откройте сайт GitHub Pages.
2. В блоке `Google Sheets` вставьте URL Web App.
3. Введите логин `teacher1` и пароль `123456`.
4. Нажмите `Войти`.

После входа импорт учеников, импорт стихов, добавление стихов и оценки будут сохраняться в Google Sheets.

## Как зарегистрировать нового учителя

На сайте в блоке `Регистрация учителя` заполните:

- ФИО учителя
- новый логин
- новый пароль
- админ-пароль `admin123`

После регистрации учитель сможет войти со своим логином и паролем.

## Как сменить админ-пароль

В Apps Script добавьте временную функцию:

```javascript
function changeAdminPassword() {
  setAdminPassword("noviy-parol-admina");
}
```

Запустите ее один раз через `Run`. После этого для регистрации используйте новый админ-пароль.
